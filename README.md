A static portfolio and knowledge graph built to track and visualize Google Cloud certification and skill data. The project leverages an automated synchronization pipeline to extract data from a public Google Cloud Skills Boost profile and render it via Astro, D3.js, and Three.js.

## Architecture

The system operates strictly as a static site, eliminating the need for server-side rendering or on-the-fly scraping. 

1. **Data Pipeline (`scripts/sync-profile.mjs`)**: A Node.js orchestration script that parses public profile HTML, extracts badge data, normalizes media assets, and infers associated technology clusters.
2. **Data Layer**: Astro Content Collections serve as the underlying data store, dynamically tracking the JSON schemas generated by the pipeline.
3. **Frontend Presentation**: Astro for static generation, React for interactive components, and Tailwind CSS for styling. Includes an environment-based CSS variables system for theming.
4. **Visualization**: A 3D force-directed graph (`react-force-graph-3d`) mapping relationships between distinct badges, categories, and technologies.

## Local Setup

### Prerequisites
- Node.js >= 20.0.0

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the synchronization pipeline to fetch current badge data:
   ```bash
   npm run sync
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

## Automated Deployment

The application is designed for zero-maintenance automated deployment via Cloudflare Pages and GitHub Actions.

1. **Continuous Data Synchronization**: The `.github/workflows/daily-sync.yml` workflow executes on a daily cron schedule. It runs the `npm run sync` script and commits any new badge JSON or image files directly back to the repository.
2. **Static Edge Deployment**: Cloudflare Pages detects the automated commit to the `main` branch, executes the build command (`npm run sync && npm run build`), and deploys the generated static assets from the `dist/client` directory to the edge.

## Technical Stack

- **Framework**: Astro
- **Components**: React 18
- **Styling**: Tailwind CSS v4, Framer Motion
- **Data Visualization**: D3-force-3d, Three.js
- **Deployment Adapter**: Cloudflare Pages (`@astrojs/cloudflare`)
